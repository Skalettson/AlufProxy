## 2. Описание программного обеспечения

**AlufProxy** — это программное обеспечение для создания зашифрованных сетевых соединений, включающее:
- Telegram-бот для управления ключами доступа
- Серверную часть на базе XRay VLESS Reality
- Клиентское приложение для Windows

Программное обеспечение предназначено для:
- Защиты приватности интернет-соединений
- Обхода необоснованных сетевых блокировок
- Тестирования сетевой безопасности

## 6. Сторонние сервисы

### 6.1. Telegram-бот

Использование Telegram-бота подразумевает:
- Согласие с Условиями использования Telegram
- Понимание, что бот размещён на сторонней платформе (Vercel)
- Принятие рисков, связанных с хранением данных в SQLite

### 6.2. VPS-серверы

При аренде VPS для серверной части:
- Вы самостоятельно выбираете провайдера
- Вы несёте ответственность за настройку сервера
- Вы соблюдаете условия хостинг-провайдера
- Разработчики не оказывают техническую поддержку сервера

### 6.3. Стороннее ПО

Программное обеспечение использует:
- **XRay-core** (лицензия MIT)
- **aiogram** (лицензия MIT)
- **customtkinter** (лицензия MIT)
- Другие библиотеки с открытым исходным кодом

Вы обязаны соблюдать лицензии стороннего ПО.

## 7. Сбор и использование данных

### 7.1. Локальные данные

Клиентское приложение хранит локально:
- Ключи доступа (в зашифрованном виде)
- Настройки подключения
- Логи работы (опционально)

Эти данные **не передаются** разработчикам.

### 7.2. Данные бота

Telegram-бот хранит в базе данных:
- Telegram ID пользователей
- Даты регистрации и окончания подписки
- Сгенерированные ключи доступа

Эти данные **не передаются третьим лицам**, за исключением случаев, предусмотренных законом.

### 7.3. Логи сервера

Серверная часть может вести логи:
- Времени подключений
- Объёма переданных данных
- IP-адресов (опционально)

Настройка логирования осуществляется администратором сервера.

**См. также:** [Политика конфиденциальности](PRIVACY_POLICY.md)
